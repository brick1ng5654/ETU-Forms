# Описание файлов конструктора

## Основные компоненты
- путь: client/pages
- файл: builder.tsx
- описание: Общая логика,Управление состоянием форм и полей, Навигация между формами, Импорт/экспорт JSON


путь: client/src/components/form-builder
- файл: FormCanvas.tsx
- описание:холст для построения форм, отображение формы в режиме редактирования, редактируемый заголовок и описание, зона для перетаскивания полей

путь: client/src/components/form-builder
- файл: PropertiesPanel.tsx
- описание:панель свойств, редактирование параметров выбранного поля, настройка валидации, опций, правильных ответов, управление опциями
